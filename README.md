# نظام إدارة الصيدلية المتكامل (Smart Pharmacy System)

هذا نظام متكامل لإدارة الصيدليات تم بناؤه باستخدام **Python** و **Django** مع واجهة عصرية تعتمد على **Bootstrap 5**.

## المميزات الرئيسية
1. **إدارة الأدوية (Inventory):** تتبع كامل للأدوية، الأسعار، الكميات، وتواريخ الانتهاء.
2. **نقطة البيع (POS):** واجهة سريعة للبيع تدعم البحث الفوري والباركود.
3. **التنبيهات الذكية:** تنبيهات آلية عند نقص المخزون أو قرب انتهاء صلاحية الأدوية.
4. **المحاسبة والتقارير:** تقارير مبيعات يومية، تحليل الأرباح، والأدوية الأكثر مبيعاً.
5. **إدارة المستخدمين:** نظام صلاحيات (مدير، صيدلي، كاشير).
6. **واجهة عصرية:** تصميم متجاوب (Responsive) يدعم اللغة العربية بشكل كامل (RTL).

## كيفية التشغيل
1. تأكد من تثبيت **Python** على جهازك.
2. قم بتثبيت المكتبات المطلوبة:
   ```bash
   pip install -r requirements.txt
   ```
3. قم بإنشاء قاعدة البيانات:
   ```bash
   python manage.py makemigrations
   python manage.py migrate
   ```
4. أنشئ حساب المدير (Admin):
   ```bash
   python manage.py createsuperuser
   ```
5. شغل النظام:
   ```bash
   python manage.py runserver
   ```
6. افتح المتصفح على: `http://127.0.0.1:8000`

## البنية التقنية
- **Backend:** Django 4.x
- **Frontend:** HTML5, CSS3, JS, Bootstrap 5 (RTL)
- **Database:** SQLite (يمكن تغييرها لـ PostgreSQL بسهولة)
- **Icons:** Font Awesome 6
